⚠️ WMS機能について:
WMS機能を利用するには、QGISでプラグインを最新バージョンに更新し、再起動してください。
現在は既存のパーマリンク機能のみが利用可能です。

🌐 利用可能なエンドポイント（既存機能）

1. インタラクティブ地図

OpenLayersを使用したインタラクティブなウェブマップを表示します。

http://localhost:8089/qgis-map?lat=35.681&lon=139.767&scale=25000

2. PNG画像直接配信

指定された座標とスケールで地図画像をPNG形式で直接取得します。

http://localhost:8089/qgis-png?lat=35.681&lon=139.767&scale=25000&width=800&height=600

3. 画像埋め込みHTML

地図画像が埋め込まれたHTMLページを表示します。

http://localhost:8089/qgis-image?lat=35.681&lon=139.767&scale=25000

📋 パラメータ説明

パラメータ 説明 例
lat 緯度（WGS84） 35.681
lon 経度（WGS84） 139.767
scale 地図スケール 25000
width 画像幅（PNG用） 800
height 画像高さ（PNG用） 600

🔧 WMS機能を有効化する方法

  1. QGISを完全に終了してください
  2. QGISを再起動してください
  3. QMapPermalinkプラグインを有効化してください
  4. HTTPサーバーが起動することを確認してください
  5. このページを再読み込みして、WMS機能をテストしてください
